# 英会話スクール LP

## 概要
HTML/CSSで制作した英会話スクールのランディングページです。

## 使用技術
- HTML5
- CSS3（Flexbox・CSS Grid・メディアクエリ）

## 対応デバイス
- PC（1100px以上）
- タブレット（768px）
- スマートフォン（375px）

## 制作のポイント
- Flexboxでヘッダー・ヒーローセクションを実装
- CSS Gridで特徴・コース・口コミの3列レイアウトを実装
- メディアクエリでスマホ対応（768px以下で1列に切り替え）
